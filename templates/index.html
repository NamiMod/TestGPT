TestGPT - Test Generation By LLM

How Does it Work?

Basically, you need to write a prompt. Based on the prompt you provide and the project you add, our test generator will automatically analyze the objects in your scene and try to write a test using the Arium Framework for the situation you described.

Please select your YAML file (Scene file) of your project, and then you can write a prompt to generate and test your selected scene with the inserted prompt.
